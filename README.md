# layerzero-rfp-checker
## Установка
`pip install -r 'requirements.txt'`

Адреса кошельков вставляем в `user_data/evm.txt`

Для запуска `main.py`

Для просмотра результатов лучше использовать `Notepad++`. Обычный блокнот некорректно отображает таблицу

## Возможности
